    <h2>Penyebab</h2>
    <ul>
      <li>Tingginya risiko kesalahan penulisan dan kehilangan data.</li>
      <li>Proses pencarian data lama sangat lambat.</li>
      <li>Keterbatasan ruang penyimpanan fisik untuk buku catatan.</li>
      <li>Sulit dalam membuat laporan cepat dan akurat.</li>
      <li>Kemungkinan catatan rusak karena faktor lingkungan (air, api, rayap).</li>
    </ul>
  </div>  <div class="card">
    <h2>Solusi</h2>
    <ul>
      <li>Menggunakan sistem pencatatan berbasis komputer atau aplikasi.</li>
      <li>Menerapkan database digital untuk penyimpanan dan pencarian cepat.</li>
      <li>Melakukan pelatihan staf agar terbiasa menggunakan sistem digital.</li>
      <li>Menyediakan cadangan (backup) data secara berkala untuk keamanan.</li>
      <li>Mengintegrasikan sistem laporan otomatis untuk mempercepat analisis.</li>
    </ul>
  </div>  <div class="card">

Solusi ini bukan hanya meningkatkan efisiensi, tetapi juga mendukung pengelolaan data yang lebih modern dan profesional.</p>
